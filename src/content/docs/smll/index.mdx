Integrate Smll in your app, Smll is a simple link shortener that allows you to share links with others.

# Getting Started

To get started, you need to create an account on Smll. Once you have an account, you need to get your API key. To get your API key, follow these steps:

1. Go to the Smll website.
2. Click on the "Account" button in the top right corner of the screen.
3. Scroll down to "Integrations" and generate a new API key.
4. Copy the API key and provide the API key in the request header `Authorization` as `Bearer <your-api-key>`.
