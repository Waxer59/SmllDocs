Integrate SmllClip in your app, SmllClip is a easy to use, app to share code snippets and documents with others.

# Getting Started

To get started, you dont need an API key. You can simply use the SmllClip app to share your code snippets and documents with others.

# Features

- Share code snippets and documents with others
- Share documents with others
